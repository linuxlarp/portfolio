# linuxlarp.dev
![Built With NextJS](https://img.shields.io/badge/built_with-NextJS-black?style=for-the-badge)
![Built With ❤️](https://img.shields.io/badge/built_with-❤️-black?style=for-the-badge)

An all-in-one NextJS portfolio website for linuxlarp. The website features an interactive CLI emulator, commands, and a terminal theme with proper image and link display.

Visit it at [linuxlarp.dev](https://linuxlarp.dev).

## Try it out
Besides the default commands, you should try out these commands:
- `coffee`
- `fastfetch`
- `secret`
- `hardware`
